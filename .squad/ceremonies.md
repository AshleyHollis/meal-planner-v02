# Ceremonies

## Manual Ceremonies

### Spec Review
- **when:** manual or after a new feature spec is drafted
- **facilitator:** Kirk
- **participants:** Spec, Scotty, Uhura, Spock, Sulu, McCoy
- **purpose:** Validate scope, architecture, dependencies, and acceptance criteria before implementation begins.

### Architecture Review
- **when:** manual for cross-cutting technical decisions
- **facilitator:** Kirk
- **participants:** Spec, Scotty, Uhura, Spock, Sulu
- **purpose:** Align APIs, data boundaries, AI behavior, and delivery sequence.

### Retro
- **when:** manual after major milestones
- **facilitator:** Kirk
- **participants:** Whole team
- **purpose:** Capture learnings and update team practices.

## Auto Triggers
- New project setup starts with the Spec constitution interview.
